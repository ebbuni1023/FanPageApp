## Technologies
Project is created with:
* React-native
* Firebase
	
## Setup
To run this project, install it locally using react-native:

```
$ cd ./FanPageApp
$ yarn install or npm install
$ npx react-native start
and open another terminal and do below command
$ npx ract-native run-ios
```
